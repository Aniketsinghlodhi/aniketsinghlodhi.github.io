## рҹ§¬ About This Project

> **A developer portfolio that doesn't just list skills вҖ” it *embodies* them.**

This is my personal portfolio website вҖ” a dark, futuristic, cyber-themed showcase of my work as a **Full-Stack Developer** and **Blockchain Engineer**. Every pixel is intentional. Every animation serves a purpose. Built from scratch with zero frameworks, zero templates вҖ” just raw HTML, CSS, and JavaScript engineered to impress.

<div align="center">

| вҡЎ Feature | рҹ”Қ Description |
|:---|:---|
| **Dark Cyber Aesthetic** | Deep space color palette with neon cyan (`#00f0ff`) and electric purple (`#a855f7`) accents |
| **Glass Morphism UI** | Translucent frosted-glass cards with layered depth effects |
| **Ambient Particles** | Floating particle system creating a living, breathing background |
| **Typing Animation** | Dynamic hero text cycling through roles and capabilities |
| **Live GitHub Stats** | Real-time repository count, followers, stars, and language breakdown via GitHub API |
| **Scroll Reveal** | Intersection Observer-powered section animations on scroll |
| **Mobile-First Design** | Fully responsive across all breakpoints вҖ” desktop to mobile |
| **Accessibility Built-In** | Skip links, ARIA labels, semantic HTML, keyboard navigation |

</div>
